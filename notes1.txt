Python is a high-level, general-purpose, interpreted programming language that is known for its simple syntax and readability.
It supports multiple programming paradigms such as procedural, object-oriented, and functional programming.

Key features of Python
  1.Interpreted Language
 	  Code is executed line by line using the Python interpreter.

  3.Dynamically Typed

  6.Supports Multiple Paradigms
